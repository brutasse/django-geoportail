Requirements: a working installation of GeoDjango and a spatial database.

Installation, detailed version:

http://packages.python.org/django-geoportail/installation.html

Installation, long story short:

 * Put the ``geoportal`` subfolder somewhere in your python path:
   * pip install django-geoportail, or
   * python setup.py install
 * Add ``geoportal`` to your ``INSTALLED_APPS`` (for templates)
 * Set ``GEOPORTAL_API_KEY`` in your ``settings.py`` to the API key you got
   here: https://api.ign.fr/geoportail/registration.do

See the online docs for detailed instructions.
